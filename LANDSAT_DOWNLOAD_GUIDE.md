## 1. Регистрация в USGS

### Шаг 1.1: Создание аккаунта
1. Перейдите на сайт: **https://ers.cr.usgs.gov/register/**
2. Заполните форму регистрации:
   - Username (имя пользователя)
   - Password (пароль)
   - Email
   - Personal Information (имя, страна)
3. Подтвердите email через письмо от USGS
4. Войдите в систему

**Важно:** Регистрация бесплатная и занимает 2-3 минуты.

## 2. Поиск снимков через EarthExplorer

### Шаг 2.1: Переход на EarthExplorer
1. Откройте: **https://earthexplorer.usgs.gov/**
2. Нажмите **Login** в правом верхнем углу
3. Введите созданные учетные данные

### Шаг 2.2: Указание координат территории
1. Во вкладке **"Search Criteria"**:
   - Выберите метод: **"Lat/Long"**
   - Введите координаты:
     - **Latitude (широта):** 58.5
     - **Longitude (долгота):** 97.5
   - Или используйте интерактивную карту для выбора области

2. Установите временной диапазон:
   - **Start Date:** 2019-06-01
   - **End Date:** 2023-08-31

3. Нажмите **"Data Sets"** внизу

### Шаг 2.3: Выбор коллекции Landsat
1. В разделе **"Data Sets"**:
   - Разверните **"Landsat"**
   - Разверните **"Landsat Collection 2 Level-2"**
   - Установите галочку на:
     - ✅ **Landsat 8-9 OLI/TIRS C2 L2**

2. Нажмите **"Additional Criteria"** (опционально):
   - **Cloud Cover:** установите максимум 10%
   - Это отфильтрует снимки с большой облачностью

3. Нажмите **"Results"**

### Шаг 2.4: Фильтрация результатов
1. В результатах отобразятся доступные снимки
2. Для каждого года (2019-2023) найдите снимок:
   - За летний период (июнь-август)
   - С минимальной облачностью
   - Проверьте превью, нажав на иконку 👁️

3. Обратите внимание на:
   - **Date Acquired** - дата съемки
   - **Cloud Cover** - процент облачности
   - **Path/Row** - номер орбиты

## 3. Скачивание данных

### Шаг 3.1: Загрузка файлов
Для каждого выбранного снимка:

1. Нажмите на иконку **скачивания** (⬇️ или дискета)
2. В появившемся окне выберите:
   - **"Landsat Collection 2 Level-2 Product Bundle"** 
   - Это полный пакет со всеми каналами и метаданными
3. Нажмите **"Download"**
4. Файл будет загружен как `.tar` архив (обычно ~1-2 GB)

### Шаг 3.2: Альтернативный метод (отдельные каналы)
Если нужны только конкретные каналы:

1. Нажмите **"Download Options"**
2. Выберите только нужные файлы:
   - ✅ `*_SR_B4.TIF` (RED - красный канал)
   - ✅ `*_SR_B5.TIF` (NIR - ближний ИК)
   - ✅ `*_QA_PIXEL.TIF` (маска облаков, опционально)
   - ✅ `*_MTL.txt` (метаданные)

### Что скачивать для каждого года:

| Год  | Период съемки | Количество файлов |
|------|---------------|-------------------|
| 2019 | Июнь-Август   | 1 архив или 2-3 файла |
| 2020 | Июнь-Август   | 1 архив или 2-3 файла |
| 2021 | Июнь-Август   | 1 архив или 2-3 файла |
| 2022 | Июнь-Август   | 1 архив или 2-3 файла |
| 2023 | Июнь-Август   | 1 архив или 2-3 файла |

**Итого:** 5 архивов (по одному на год)

## 4. Организация файлов

### Шаг 4.1: Распаковка архивов
1. Создайте папку `landsat_data/` в директории проекта
2. Распакуйте каждый `.tar` архив
3. Организуйте файлы по годам

### Шаг 4.2: Рекомендуемая структура папок

```
GEOML/project/
├── project.ipynb
└── landsat_data/
    ├── 2019/
    │   ├── LC08_LXXX_RXXX_XXXXXXXX_XXXXXXXX_02_T1_SR_B4.TIF
    │   ├── LC08_LXXX_RXXX_XXXXXXXX_XXXXXXXX_02_T1_SR_B5.TIF
    │   └── LC08_LXXX_RXXX_XXXXXXXX_XXXXXXXX_02_T1_MTL.txt
    ├── 2020/
    │   ├── LC08_..._SR_B4.TIF
    │   └── LC08_..._SR_B5.TIF
    ├── 2021/
    ├── 2022/
    └── 2023/
```

### Шаг 4.3: Важные файлы

Каждый архив Landsat содержит множество файлов. Для проекта нужны **только**:

- `*_SR_B4.TIF` - Красный канал (RED, 640-670 нм)
- `*_SR_B5.TIF` - Ближний ИК канал (NIR, 850-880 нм)

Остальные файлы можно удалить для экономии места или оставить для дополнительного анализа.

## 5. Автоматическая загрузка через Python

### Вариант для продвинутых пользователей

Если нужно скачать много снимков, можно автоматизировать процесс:

### Шаг 5.1: Установка библиотеки

```bash
pip install landsatxplore
```

### Шаг 5.2: Поиск снимков

```python
from landsatxplore.api import API

username = "ваш_usgs_username"
password = "ваш_usgs_password"

api = API(username, password)

scenes = api.search(
    dataset='landsat_ot_c2_l2',
    latitude=58.5,
    longitude=97.5,
    start_date='2019-06-01',
    end_date='2023-08-31',
    max_cloud_cover=10
)

print(f"Найдено {len(scenes)} сцен")

for scene in scenes[:5]:
    print(f"{scene['acquisition_date']} - {scene['cloud_cover']}% облачность")

api.logout()
```

### Шаг 5.3: Скачивание

```python
from landsatxplore.earthexplorer import EarthExplorer
import os

ee = EarthExplorer(username, password)

os.makedirs('landsat_data', exist_ok=True)

for scene in scenes[:5]:
    print(f"Скачивание: {scene['display_id']}")
    ee.download(scene['entity_id'], output_dir='landsat_data')

ee.logout()
```

**Внимание:** Скачивание через API может занять много времени (каждый архив ~1-2 GB).

## Проверка загруженных данных

После скачивания проверьте:

1. **Количество файлов:** Должно быть минимум 2 файла на год (B4 и B5)
2. **Размер файлов:** Каждый `.TIF` файл обычно 50-200 MB
3. **Формат:** Убедитесь, что файлы с расширением `.TIF` или `.tif`

### Тестовая ячейка в notebook:

```python
import glob

for year in [2019, 2020, 2021, 2022, 2023]:
    b4_files = glob.glob(f'landsat_data/{year}/**/*_SR_B4.TIF', recursive=True)
    b5_files = glob.glob(f'landsat_data/{year}/**/*_SR_B5.TIF', recursive=True)
    print(f"{year}: B4={len(b4_files)}, B5={len(b5_files)}")
```

## Частые вопросы

**Q: Сколько места занимают данные?**  
A: Полные архивы: ~5-10 GB. Только B4 и B5: ~500-1000 MB.

**Q: Как долго скачивается?**  
A: Зависит от интернета. Обычно 10-30 минут для всех архивов.

**Q: Что делать, если нет снимков с низкой облачностью?**  
A: Увеличьте порог облачности до 20-30% или выберите другую дату.

**Q: Можно ли использовать Landsat 7 вместо 8?**  
A: Да, но Landsat 7 имеет проблемы с SLC (полосы на изображениях с 2003 года).

**Q: Где посмотреть пример результата?**  
A: В notebook есть режим демо-данных - просто запустите код без скачанных файлов.
